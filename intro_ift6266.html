Personal research journal: deep learning for speech synthesis

Jan 25, 2014

This is the introduction to a series of reports on my experiments on deep learning methods for speech synthesis. These experiments are part of my coursework for Dr. Yoshua Bengio's Representation Learning course at Université de Montréal. All the related code is going to be posted at a GitHub repository as well.

Please visit the ift6266 tag page for a list of all the posts related to this project.
